Hawknee

by Semahawk
Easily implement a forum to your Ruby on Rails 3 application!

Development

According to Semantic Versioning the major version zero (0.y.z) should not be considered stable. There will be a lot of changes unitll major version 1 (1.y.z). So, Hawknee is unstable and not-working, unless you see version 1.y.z is released. Before version 1.y.z, anything can change, so please be patient. Oorah?

Dependencies

  • rails >= 3.0.0

Installation

$ gem install hawknee              # Install Hawknee
$ cd /path/to/your/rails/app/path  # Go to your Rails application
$ edit Gemfile                     # Put there: gem 'hawknee', 'x.y.z'
$ rails generate hawknee:install   # Copy all the needed files. You can use 'g' instead of 'generate'
$ rails server                     # Run the server.

Download

You can download this project in either zip or tar formats, and install by typing: 

$ cd /path/to/hawknee-x.y.z.gem
$ gem install hawknee-x.y.z.gem

You can also clone the project with Git by running: 

$ git clone git://github.com/semahawk/hawknee
